#BMI Calculator
## Project Description
This is a simple BMI(Body Mass Index) Calculator developed using python. The program takes the user's weight in kilograms and height in meters, calculates the BMI, and displays the corresponding health category.

## Features
-Accepts weight in kilograms
-Accepts height in meters
-Calculates BMI using the formula: BMI = weight / (height ** 2)
-Displays BMI rounded to two decimal places 
-Classifies BMI as:
 -Underweight
 -Normal
 -Overweight
 -Obese
-Handles invalid numeric input
-Rejects negative or zero values

## Technologies used
-Python

# How to Run
1. Open the project folder in VS Code
2. Open the terminal
3. Run the following command:
python bmi_calculator.py
4. Enter your weight in kilograms
5. Enter your height in meters
6. The BMI and health category will be displayed

## Example
Weight: 60kg
Height: 1.65 m
BMI: 22.04
Category: Normal
