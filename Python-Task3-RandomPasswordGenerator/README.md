# Random Password Generator

## Project Description
This is a Random Password Generator developed using Python. The program msecure random passwords based on user-selected character types and password length.

## Features 
-User can choose the desired password lentgh 
-Minimum password lentgh is 8 characters 
-Supports uppercase letters
-Supports lowercase letters
-Supports numbers
-Supports symbols
-Requires at least two character types
-uses Python's secrets module for secure password generation
-Handles invalid password length input
-Allows users to generate multiple passwords

## Technologies Used
-Python
-secrets module
-string module

##  How to Run
1. Open the project folder in VS Code
2. Open the terminal
3. Run the following command:
python password_generator.py
4. Enter the desired password length.
5. The generated password will be displayed.
