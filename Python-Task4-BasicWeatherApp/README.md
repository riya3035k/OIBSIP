# Basic Weather App
## Project Description
This is a Basic Weather App developed using Python. The application fetches real-time weather information for a city using the OpenWeatherMap API.

## Features
-Accepts a  city name from the user
-Fetches real-time weather data
-Displays temperature in Celsius
-Displays humidity
-Displays weather description
-Handles invalid city names or API errors

## Technologies Used
-Python
-Requests Library
-OpenWeatherMap API

## How to Run
1. Install the requests library:
pip install requests
2. Get an API key from OpenWeatherMap.
3. Add your API key in the python file:
API_KEY = "YOUR_API_KEY'
4. Run the program:
python weather_app.py
5. Enter the city name when prompted
